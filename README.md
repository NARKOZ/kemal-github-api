# Kemal Github API

Mock Github API implemented with Kemal.

## Installation

```
git clone https://github.com/sdogruyol/kemal-github-api
cd kemal-github-api
crystal build --release src/kemal-github-api.cr -o bin/kemal-github-api
./bin/kemal-github-api
```
